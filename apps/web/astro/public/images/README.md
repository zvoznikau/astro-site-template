# Images Directory

This directory contains images used in the landing page.

## Required Images

### `hero-app-mockup.svg` (or `.png`)
- **Location**: Hero section
- **Size**: 400x711px (9:16 aspect ratio for mobile mockup)
- **Format**: SVG (preferred) or PNG
- **Content**: App screenshot showing server monitoring dashboard
- **Fallback**: SVG placeholder is included

### `feature-pattern.svg`
- **Location**: Feature cards (decorative background)
- **Size**: 128x128px
- **Format**: SVG
- **Content**: Decorative circular pattern
- **Usage**: Subtle background element

## Adding Your Own Images

1. Replace `hero-app-mockup.svg` with your actual app screenshot
2. Recommended formats: SVG (scalable) or PNG (high quality)
3. Optimize images before adding (use tools like ImageOptim, Squoosh, etc.)
4. Maintain aspect ratios for best display

## Image Optimization

- Use SVG for illustrations and icons
- Use WebP or optimized PNG for photos
- Compress images to reduce file size
- Use appropriate dimensions (don't use oversized images)
